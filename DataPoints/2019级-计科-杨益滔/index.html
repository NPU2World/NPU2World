    2019级-计算机学院-杨益滔 (CUHK)

    基本背景

    本科专业： 计算机学院-计算机科学与技术

    GPA&排名： 3.85+， 11/200+

    奖学金情况： 优秀大学生标兵 (校特等奖学金) + 国家奖学金

    科研情况： 申请时一篇c类一作再投（最后被拒），现转投IWQoS'23

    标化成绩： IELTS 7.0

    申请方向： System and Network

    最终去向： CSE PHD@CUHK

    联系方式： 2021614317 (QQ)

    邮箱： yangyitao200107@outlook.com

    申请时间线

    1. 前两年过得比较浑浑噩噩，一直在内卷GPA和出国两条路上来回纠结。大三上学期过了Mit6.S081，Stanford CS144之后逐渐有了放弃保研，申请出国的想法。
    2. 大三下的4月可以开始申请CUHK提前批，提交材料。
    3. 4月，为了准备陶瓷，特意找了几个大厂投实习简历（主要是练练手，熟悉熟悉不同面试风格），当时面了阿里，字节，dolphindb。准备面试的同时也继续复习os，network，db相关的底层知识。
    4. 5月底通过cuhk的提前批面试，之后开始陶瓷。第一个套的就是现在的老板，因为崔老师和现在老板之间有些connection，同时之前有学长在老板的组里，和学长提前聊了聊组里的情况。正式陶瓷时，老师反馈非常积极，当天晚上就约了面试，之后不久就发了口头offer。

    标化准备及建议

    IELTS总分7.0 (阅读7.5 听力8.0 写作6.0 口语7.0)，由于大一就考过一次雅思，因此备考没什么压力，主要就是刷题、看公开资源和找外教练口语。

    阅读听力： 刷剑桥雅思系列

    写作和口语: 可以参考b站Group9120系列视频，链接如下：口语备考 写作备考

    找外教： Free talk dream寻梦教育，推荐找欧美，非欧美的外教（我当时找的是一个伊朗人，特别耐心，能听出很多我的小问题，大约一共上了10节课不到，最后口语7.0）

    本校实验室/导师/项目经历

    在大二的时候，和jjx, wth, ssd一起参加了龙芯杯。比赛的内容是设计一个MIPS-CPU，并在CPU上搭建起一个小型的Soc，最终启动LINUX操作系统。在这个项目让我对于System有了初步的认识。最终靠着大家的通力协作，最终我们比赛的结果相当不错。

    大三的时候和zb一起加入了崔禾磊老师的课题组，崔老师人非常nice，很愿意带本科生，之后在崔老师指导下写了一篇短文投给了Globecom'22，可惜最后被reject。暑假接着完善这篇工作，现在在改投IWQoS'23。

    我的故事/申请季经验总结

    回顾我的前三年，参加龙芯杯和加入崔老师课题组是对我本科阶段改变最大的两件事，龙芯杯让我找到了感兴趣的方向(System)，而在崔老师课题组内让我开始科研探索，并最终坚定了出国的打算。

    为什么要出国？

    其实我觉得大多数人对于为什么出国这一问题都是人云亦云，随波逐流，只是认为出国是保研/考研以外的另一条道路。我想学弟学妹们在做出出国的决定之前，应当真正问问自己“我为什么要出国？”

    大二学年，学院每学年的GPA、综测评定和保研带来的压力让我喘不过气。学习每一门课程，参加每一项比赛之前考虑的不是我能学到什么东西，我是否对这门课程真的感兴趣，而是比赛能给我带来多少综测加分？在这种有些功利性的评价体系下的三年让我非常疲倦，尤其是大二以后，我根本不知道某些课程&很水的比赛的意义在哪。同时在学习知识太过急功近利，我也找不到内心对于获取知识真正的充实和快乐，每项技术都至停留在会用，大概知道怎么用，谈及细节而不知道其底层原理。空有一纸还算可以的GPA成绩，但谈及自己能力，总会有种深深的无力感。

    在我打完了龙芯杯比赛，以及后续接触到了MIT, Stanford和CMU相关的System相关课程，我第一次感受到震撼。公开的课程资料，视频资源以及开放的实验评阅系统，让我收获很多。并逐渐对系统方向产生兴趣。大三学年加入了崔老师的课题组之后，在崔老师的带领下走进了科研的大门，同时也进一步了解到了国内国外科研氛围的差异，并最终坚定的出国的想法。

    套磁经验分享

    我算是凭借connnection推荐的，因此我想对于志在出国读博的同学们来说，利用好本科阶段找到自己真正感兴趣的方向，联系本校老师接触科研，最终有一定的科研成果，获得推荐信/点对点的推荐，在出国申请中是一项很大的加分项。

    选校策略分享

    对于PHD而言，选校实际上就是选研究组：

    1. 组里的方向主要是什么？自己是否真的有兴趣？
    2. 能不能正常毕业？毕业后出路如何？
    3. 组里的Publication一般发表在哪些会议？

    给后来者的建议

    • 利用本科阶段多尝试不同方向，多试错，不要害怕失败，找到自己真正兴趣所在，毕竟兴趣是最好的导师！

    • 学习有意义的课程，参加有挑战性的比赛，不要为了GPA的小数点后几位而过度焦虑。

    • 联系本校导师，在老师的指导下接触科研，多花时间看文献，定期主动和老师meeting，老师是不会拒绝一个self-motivated的本科生。

    • 如果申请时被全聚德，还可以在HK做RA过渡，RA每个月工资不低，同时也可以搞research攒文章，这算是最后的出路。

    • 多多锻炼，身体才是革命的本钱hhhhh。
